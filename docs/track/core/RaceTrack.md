# RaceTrack

## Описание

RaceTrack описывает конфигурацию гоночной трассы: точку появления игрока, набор чекпоинтов и количество кругов по умолчанию.

## Основные поля инспектора

* playerSpawn — Transform, куда ставится машина игрока при старте сцены.
* checkpointsRoot — родительский объект, внутри которого лежат все объекты типа Checkpoint.
* checkpoints — отсортированный список чекпоинтов; элемент с index = 0 используется как старт/финиш.
* defaultLaps — количество кругов по умолчанию для этой трассы.

**Важно:** если checkpoints пуст, в Awake будет выведена ошибка в консоль, а если первый чекпоинт не имеет index = 0, появится предупреждение о неверной конфигурации старт/финиша.

## Жизненный цикл

При запуске сцены Awake:

* Собирает дочерние Checkpoint из checkpointsRoot, если список ещё не заполнен.
* Сортирует их по полю index.
* Проверяет базовую корректность конфигурации.

OnDrawGizmos рисует:

* зелёную сферу и стрелку в месте playerSpawn, чтобы было видно направление старта;
* жёлтые wire-кубы в позициях всех чекпоинтов, используя их размер GizmoSize.
